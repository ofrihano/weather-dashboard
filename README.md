A Python application that fetches weather data from OpenWeatherMap API and provides:
- Current weather conditions for multiple cities
- Weather forecasts
- Historical trends
- Temperature alerts
- "Best day this week" recommendations

## Setup

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Create a `.env` file with your OpenWeatherMap API key
4. Run: `python main.py`

## Features
- [ ] Current weather display
- [ ] 5-day forecast
- [ ] Temperature alerts
- [ ] Best day recommendation
- [ ] Multi-city tracking

## API
Uses [OpenWeatherMap API](https://openweathermap.org/api) (Free tier)
